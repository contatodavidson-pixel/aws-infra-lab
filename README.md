## 📖 Sobre o Projeto

Este projeto demonstra a implementação de uma infraestrutura em nuvem utilizando boas práticas de **DevOps** e **Cloud Computing**.

O objetivo é simular um ambiente real com foco em:

* Alta disponibilidade
* Segurança
* Escalabilidade
* Automação de infraestrutura

---

## 🧠 Arquitetura

📌 Diagrama da solução:

```
Usuário → Internet → Load Balancer → EC2 → Aplicação
                              ↓
                           Banco (RDS)
```

---

## ☁️ Serviços AWS utilizados

* EC2 (servidores)
* S3 (armazenamento)
* RDS (banco de dados)
* IAM (controle de acesso)
* VPC (rede)
* CloudWatch (monitoramento)

---

## ⚙️ Tecnologias e Ferramentas

* Terraform (Infraestrutura como código)
* Git & GitHub
* Linux (Ubuntu)
* Bash / Shell Script

---

## 🚀 Deploy

```bash
git clone https://github.com/contatodavidson-pixel/aws-infra-lab.git

```
